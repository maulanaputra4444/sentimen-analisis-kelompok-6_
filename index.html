ℹ️ Tentang SentiMind

Analisis Sentimen Komentar Media Sosial — Kelompok 6

📌 Tentang Proyek

SentiMind adalah aplikasi analisis sentimen komentar media sosial berbahasa Indonesia yang dikembangkan oleh Kelompok 6. Aplikasi ini memungkinkan pengguna untuk menganalisis sentimen dari komentar media sosial secara batch, dengan dukungan hingga 10 komentar sekaligus.

Hasil analisis meliputi label sentimen (Positif, Negatif, Netral) beserta skor kepercayaan model, serta visualisasi distribusi sentimen dalam bentuk grafik interaktif.

✅ Model: Indonesian RoBERTa Base Sentiment Classifier

🛠️ Teknologi

Proyek asli dibangun menggunakan Streamlit dan HuggingFace Transformers (model w11wo/indonesian-roberta-base-sentiment-classifier). Versi web ini dibangun ulang dengan HTML/CSS/JavaScript murni, mengintegrasikan API AI untuk analisis sentimen bahasa Indonesia.
